🧾 Zoho Books — OAuth bootstrap

Deze pagina ruilt eenmalig de authorization-code uit Zoho's API Console om voor een permanente refresh-token. De credentials worden direct in de server-.env geschreven. Na succes ontvangt elke Mollie-betaling automatisch een factuur in Zoho Books.

Scopes voor nieuwe code: gebruik in api-console.zoho.com deze 5 scopes:
ZohoBooks.invoices.ALL,ZohoBooks.contacts.ALL,ZohoBooks.creditnotes.ALL,ZohoBooks.customerpayments.ALL,ZohoBooks.settings.ALL
